# Js-Learning-Path

> **Repository:** [github.com/subashbelina/JavaScript-Learning-Path](https://github.com/subashbelina/JavaScript-Learning-Path)

A **read-and-learn** JavaScript roadmap for beginners and self-taught developers. Each topic lives in its own folder with small, focused files — one concept per file, clear comments, and syntax examples.

No npm install. No project setup. Just open the files and study.

---

## Short Description

**Js-Learning-Path** takes you from core JavaScript through modern ES6+ features, React, and Next.js — step by step. Every file is written for learning: explanations at the top, code examples inside, and optional `node` output to review key points.

---

## How to Use

1. **Read** — open any `.js` file in your editor
2. **Learn** — follow the comments and code examples
3. **Practice** — try the examples yourself in `Practice folder/`
4. **Optional** — run a file to see a printed summary:
   ```bash
   node "JavaScript Fundamental/Variables/let-const.js"
   node "Modern-JavaScript/optional-chaining.js"
   node "JavaScript library /React/intro.js"
   ```
5. **DOM basics** — open `JavaScript Fundamental/DOM-basics/index.html` in a browser

## Learning Path (recommended order)

### 1. JavaScript Fundamental
Core language — start here.

| Folder | Topics |
|--------|--------|
| `Data-types/` | string, number, boolean, null, undefined, symbol, bigint, object, array, function |
| `Variables/` | let, const, var, scope, closures |
| `Operator/` | arithmetic, assignment, comparison, logical, bitwise, ternary |
| `Conditionals/` | if/else, switch, nested conditions |
| `Loops/` | for, while, do-while, for...of, for...in |
| `Function/` | declaration, expression, arrow, params, return, callback, IIFE |
| `Objects/` | creation, methods, keys/values, this, Object.assign |
| `Array-methods/` | map, filter, reduce, forEach, find, slice, splice |
| `Destructuring/` | array, object, nested & default |
| `Spread-rest/` | spread in arrays/objects, rest parameters |
| `Template-literals/` | basic & multiline templates |
| `Classes/` | basics, inheritance, static & getters |
| `JSON/` | parse & stringify |
| `Async-await/` | promises, async/await, parallel |
| `Fetch-api/` | GET & POST requests |
| `Error-handling/` | try/catch, throw, async errors |
| `ES6-modules/` | named export, default export, import |
| `DOM-basics/` | select, change, create elements, events |

### 2. Modern-JavaScript

### 3. JavaScript library / React
React syntax and concepts (read-only reference).

Components, JSX, props, state, events, hooks, context, forms, and more.

### 4. JavaScript Framework / Nextjs
Next.js App Router fundamentals (read-only reference).

Routing, layouts, server/client components, data fetching, API routes, metadata, middleware.

### 5. Practice folder
Your space to write and experiment with what you learn.
